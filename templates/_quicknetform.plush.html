This is a quick edit form. This will assume net control 5 minutes before the net, add early checkins one minute prior to net open, regular checkins 30 seconds after net open, and close the net. You can make changes and add comments afterwards. This form will be disabled if there is already recorded net activity.

<%= form_for( quicknet, {action: quicknetPath({id: net.ID}), method: "POST"}) { %> <%= if (errors) { %> <%= errors %> <% } %>
<%= f.TextAreaTag("EarlyCheckins", {rows:7}) %>
<%= f.TextAreaTag("RegularCheckins", {rows:7 }) %>
<%= f.InputTag("NetControl") %>
<%= f.DateTimeTag("Opened", {value: net.PlannedStart.Format("2006-01-02T15:04")}) %>
<%= f.DateTimeTag("Closed", {value: net.PlannedEnd.Format("2006-01-02T15:04")}) %>
<% } %>
